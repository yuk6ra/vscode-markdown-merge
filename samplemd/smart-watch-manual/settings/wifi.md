# Wi-Fi Setup

## Connecting to Wi-Fi

1. Open **Settings > Network > Wi-Fi**
2. Select your network from the list
3. Enter the password and tap **Connect**

## Troubleshooting Wi-Fi

| Problem | Solution |
|---------|----------|
| Network not found | Move closer to the router |
| Connection drops | Restart the watch |
| Slow speed | Check if other devices are connected |
